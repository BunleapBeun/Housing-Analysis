# Housing Data Analysis Project

## Housing Market Analysis & Insights
A comprehensive data analysis project exploring property characteristics, pricing patterns, and market segmentation across 545 housing records. Using Python for data processing, statistical analysis, and visualization to derive actionable insights for real estate professionals and market understanding.

## Analysis Features
- **Market Segmentation**: Price range analysis and property categorization
- **Feature Distribution**: Bedroom, bathroom, parking, and area analysis  
- **Statistical Insights**: Pricing trends and feature correlations
- **Visual Analytics**: Interactive charts and automated reporting

## Technical Stack
- Python, Pandas, NumPy for data processing
- Matplotlib, Seaborn for visualization
- Jupyter Notebook for interactive analysis
- Automated reporting scripts

## Project Structure
- `data/`: Contains the Housing.csv dataset
- `notebooks/`: Jupyter notebook for interactive analysis
- `scripts/`: Python script for automated analysis
- `output/`: Generated charts and visualizations

## Setup Instructions
1. Clone or download this project
2. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Run project with Jupyter
   ```bash
   jupyter notebook notebooks/housing_analysis.ipynb
   ```
4. Run Project with Python
   ```bash
   python scripts/housing_analysis.py
   ```
